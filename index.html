2020 October

Red Badge Project: Travel Planning App

Each student was assigned a solo full stack-development project of creating a web API in three weeks. As the final assignment of the full-time web development course at Eleven Fifty Academy, this project built upon concepts we learned in the blue badge project. Like the blue badge project, the back end was built using Node.js, PostgreSQL, and Sequelize, and the front end was made with React. The red badge project differed from the blue badge project in several key areas:

  • The back end implemented admin/user roles
  • The front end was built using React Legacy -no hooks- and TypeScript
  • Instead of Bootstrap and Reactstrap, the red badge project was styled with Material-UI and Radium
  • The red badge project featured more advanced database implementation, including the use of database associations
  • In addition to a user table, at least two tables required full CRUD functionality

I created a travel planning app. Using this app, a logged in user may search for a desired destination using the search tool in the destinations tab. The search grabs data from the OpenTripMap API that most closely matches the user's search text. The user may add the search result to a personal saved destinations list.

From the trips tab, a logged in user can create trips. A trip may consist of any number of destinations. Saved destinations may be added to a trip.

2020 September

Blue Badge Project: To-Do List

As part of a 3-person team, we were tasked with a full-stack development project of creating a web API in one week. The back end was made using Node.js, PostgreSQL, and Sequelize. The front end was made using React hooks, JavaScript, Bootstrap, and Reactstrap. Aside from the user table, at least one table required full CRUD functionality. Our team decided to make a to-do list app where users were able to register an account and login. Once logged in, the user was able to create, save, edit, or delete items on a to-do list.

2020 August

Holidays 2020 Information Search API Project

If you ever wanted to learn about holidays around the world, please try my Holidays 2020 Information Search API Project. Initially, a full list of nations populates the table. You are welcome to scroll through the full list and click on the Holiday Info Buttons to learn more about each country's holidays of 2020. For a more streamlined process, you may also type the name of the desired country in the search bar. As you type, the table will filter the results with the countries that best match your input.
