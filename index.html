about me

Phillip M. Blaine, web developer | pianist | composer

As an aspiring fledgling web developer, I wish to bring my own unique angle to the field. Throughout the development process, I strive for an easy-to-understand approach and friendly cooperation with my peers. For web design, I enjoy exploring creative ways to provide a pleasant web experience and a user-friendly interface.

Prior to embarking into the world of web development, I spent much of my life pursuing a career in music and college teaching. I am an accomplished pianist and composer with years of performance and teaching experience.

Pokémon GSC Search API Project

Need to refresh your Pokémon knowledge? If so, please try out my Pokémon GSC Search API Project! You can look up any pokémon via their english name or national dex ID number. Or, if you are feeling indecisive, try out the Random Pokémon Button. Hover over pokemon to learn about their names, types, and abilities.

Holidays 2020 Information Search API Project

If you ever wanted to learn about holidays around the world, please try my Holidays 2020 Information Search API Project. Initially, a full list of nations populates the table. You are welcome to scroll through the full list and click on the Holiday Info Buttons to learn more about each country's holidays of 2020. For a more streamlined process, you may also type the name of the desired country in the search bar. As you type, the table will filter the results with the countries that best match your input.
